# Response Contract

Default response shape:

## Recommendation

- `Category`: the best ZenKit category
- `Use`: 1-3 recommended primitives
- `Why`: one short explanation per primitive

## Minimal Skeleton

Provide a short SwiftUI snippet only when it clarifies composition.

## References

List 1-3 real repo paths, preferring:

- matching showcase screen
- component source file
- smoke test if it demonstrates composition

## Caveats

- mention native SwiftUI when appropriate
- mention deprecated ZenKit API when relevant
